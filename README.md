# podcasty
Web App for "Introduzione alle Applicazioni Web" - Politecnico di Torino
## What is podcasty
Website when you can lister and/or upload podcast but also your music if you want.

## Requirements
- flask
- flask_session
- flask_login
- datetime
- werkzeug.security
- PIL
